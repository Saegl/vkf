# Create vk app

0. Auth to any vk account
1. Go to https://vk.com/apps?act=manage, and click `create`  
   Any name  
   type standalone

2. Go to app settings

   Enable OpenAPI  
   address - http://127.0.0.1:3434  
   domain - 127.0.0.1

   Activate app
   Status - enabled and shown to everyone

3. Copy id to use with `vkf auth <id>`
